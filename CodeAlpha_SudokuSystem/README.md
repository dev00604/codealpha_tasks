🧩 Sudoku Solver (C++)

📌 Description

This project is a console-based Sudoku Solver developed in C++. It uses the Backtracking Algorithm to solve Sudoku puzzles efficiently. Users can solve a built-in puzzle or enter their own 9×9 Sudoku puzzle.

✨ Features

- Solve a built-in Sudoku puzzle
- Enter a custom Sudoku puzzle
- Uses the Backtracking Algorithm
- Displays the Sudoku grid with colored output
- Shows the number of solving steps
- Measures and displays execution time
- Handles invalid input

🛠️ Technologies Used

- C++
- Backtracking Algorithm
- Object-Oriented Programming Concepts
- STL
- Chrono Library

📂 Files

- "sudoku_solver.cpp" - Main source code

▶️ How to Run

Compile the program:

g++ sudoku_solver.cpp -o sudoku_solver

Run the program:

./sudoku_solver

(On Windows: "sudoku_solver.exe")

📋 Menu

1. Solve the built-in sample puzzle
2. Enter my own puzzle
