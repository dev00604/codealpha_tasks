# 🔐 Login and Registration System (C++)

## 📌 Description
This is a console-based Login and Registration System developed in C++. It allows users to register with a username and password, securely stores user credentials using password hashing, and provides login authentication through a file-based database.

## ✨ Features
- User Registration
- User Login Authentication
- Password Hashing (djb2 algorithm)
- File-based User Data Storage
- Username Validation
- Password Strength Validation
- Password Confirmation
- Login Attempt Limit
- Display Registered Users

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Standard Template Library (STL)

## 📂 Files
- `logic_system.cpp` - Main source code
- `users.dat` - Stores registered user information

## ▶️ How to Run

Compile the program:

```bash
g++ logic_system.cpp -o login_system
```

Run the program:

```bash
./login_system
```

(On Windows: `login_system.exe`)

## 📋 Menu
```
1. Register
2. Login
3. List Registered Users
0. Exit
```
