# 🎓 CGPA Calculator (C++)

## 📌 Description
This is a C++ console-based application that calculates a student's CGPA based on the credits and grade points of each subject. The program accepts user input for student details and displays the calculated CGPA.

## ✨ Features
- Enter student name
- Enter multiple subjects
- Input credits and grade points
- Automatically calculate CGPA
- Display student name and final CGPA

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL Vector

## ▶️ How to Run
1. Compile the program:
   ```
   g++ cgpa.cpp -o cgpa
   ```
2. Run the executable:
   ```
   ./cgpa
   ```
   (On Windows: `cgpa.exe`)
